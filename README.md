💾 Save Input Using Local Storage

📌 Overview

This project is a simple web application that allows users to save input text using the browser's Local Storage. The saved data remains even after refreshing the page, and users can also clear the stored data with a single click.

🚀 Features

* 💾 Save user input in Local Storage
* 🔄 Data persists even after page reload
* ❌ Clear stored data easily
* ⚡ Real-time interaction using JavaScript

 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap
* JavaScript (Local Storage & DOM Manipulation)


📂 Project Structure

```id="stp1a2"
├── index.html
├── saveinlocalstorage.css
├── saveinlocalstorage.js
```

▶️ How It Works

* User enters text in the input field
* Clicking **Save** stores the text in Local Storage
* The stored value is automatically displayed when the page reloads
* Clicking **Clear** removes the stored value and clears the input

📸 Output

* Input field with Save and Clear buttons
* Saved text remains after refresh
* Clear button removes stored data instantly


 🎯 Learning Outcomes

* Understanding Local Storage in JavaScript
* Storing and retrieving data using `localStorage`
* Handling button click events
* Updating UI dynamically
